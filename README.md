## OCaml Portfolio Checker

A beginner-friendly OCaml CLI (Command Line Interface) program that:
- Suggests stocks based on investment horizon
- Ensures diversification across sectors
- Allocates money evenly
- Computes overall portfolio risk
- Shows sector-wise breakdown

### Concepts Used
- Algebraic data types
- Pattern matching
- Lists and higher-order functions
- Option and Result types

### How to Run
Load the file in utop:

```ocaml
#use "portfolio_checker.ml";;
run_portfolio_checker ();;
